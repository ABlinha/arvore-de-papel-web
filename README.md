## Projeto
Esse é um projeto pessoal. 
Um site-galeria com o intuito de mostrar aos clientes Árvore de Papel os produtos disponíveis em Catálogo.

## Tecnologias
Esse projeto está sendo desenvolvido usando as seguintes tecnologias:

- Figma
- HTML
- CSS
- Git e Github

## Layout
Você pode visualizar o layout do projeto através [desse link](https://www.figma.com/file/MMs8U1UA5MLvSddC54jY7i/Galeria-%C3%81rvore-de-Papel?type=design&t=sWQ43sdXBAJ1yPAw-6).
